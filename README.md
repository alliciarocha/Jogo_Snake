# 🐍 Jogo da Cobrinha
Jogo da Cobrinha em Python utilizando Pygame

## 🛠️ Instalação
Para compilar e instalar as dependências do projeto, execute o comando:
```bash
pip install pygame
```

## 🚀 Execução
Depois de instalar as dependências, execute a aplicação com:
````bash
python nome_do_arquivo.py
````
